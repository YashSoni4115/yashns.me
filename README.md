# yashns.me

Static portfolio site with a Win98 inspired layout.

## Run locally
Open `index.html` in your browser.

## Deploy (GitHub Pages)
- Put these files in a GitHub repo
- Settings -> Pages -> Deploy from branch (main, root)
- Set Custom domain to `yashns.me`
- Keep `CNAME` in the repo root

## Resume
Replace `resume/resume.pdf` with your real resume PDF.
